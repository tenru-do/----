# tenru-do Projects

This repository is organized as a small project collection.

## Boxes

- [Rokid](rokid/README.md)
  - [Rokid Zoom Camera](rokid/rokid-zoom-camera/README.md): unofficial Rokid Glasses camera prototype with temple-swipe zoom, zoomed photo capture, and zoomed video recording.
- [Excel Macros](excel-macros/README.md)
  - Legacy Excel/VBA macro modules.

## Notes

The Rokid app is an unofficial prototype and is not endorsed by Rokid.

Build outputs, Android SDK files, screenshots, APKs, and local investigation artifacts are intentionally excluded by `.gitignore`.
